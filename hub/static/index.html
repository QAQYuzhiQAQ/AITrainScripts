格式转换

一键流程：读取来源目录 → 按所选模式处理为 PNG → 写入输出目录 → 批量重命名（可选联动 .txt / .caption）。

模式一 · 等效面积（64 倍数）：目标面积 = 宽×高，输出画布长宽均为 64 的倍数，等比缩放 + 透明边（不拉伸）。

模式二 · 固定画布：输出严格等于所填宽×高，等比缩放 + 透明边补足。

步骤 4：批量重命名（输出目录内）

目标面积（基准宽 × 基准高）表示期望的像素总面积（宽 × 高），不是强制输出为相同宽高。

程序会保持原图比例，自动选取最接近该总面积、且长宽均为 64 整数倍的画布，再透明居中放置。例如填 1024×1024 时，横图可能输出 1344×768 等尺寸。

将 PNG 等比缩放后，完整放入你指定的固定宽高透明画布并居中。输出尺寸始终等于所填画布宽 × 高（与「格式转换」的面积智能匹配不同）。

递归扫描目录，仅处理尺寸为 2560×1440 的 PNG，按固定区域 (320,0)–(2240,1440) 裁剪，并保持原有文件夹结构。

递归扫描目录，仅保留宽高与下方设定完全一致的图片，其余删除。请先预览，确认后再执行（不可逆）。

确认删除

对单个文件夹内的图片批量重命名。支持「前缀 + 补零编号」或「纯序号」两种模式，默认仅预览。
